Why Kubelens?

Powerful features for modern Kubernetes management

🌐

Multi-Cluster

Connect and manage unlimited Kubernetes clusters. Switch between them seamlessly with real-time metrics and health monitoring.

📊

Comprehensive Resources

Manage all Kubernetes resources: Pods, Deployments, Services, ConfigMaps, Secrets, CRDs, and more.

🔄

Real-Time Updates

WebSocket-based live updates for all resources. Auto-refresh with metrics from metrics-server.

🎨

Modern UI/UX

Beautiful, responsive design with dark/light themes. Resizable columns, advanced search, and sortable tables.

🔧

Full CRUD Operations

View, Edit, Create, Delete resources. YAML editor with syntax highlighting. Scale, restart, and manage workloads.

💻

Pod Shell & Logs

Interactive terminal access to pods with xterm.js. Real-time log viewing with follow mode.

📈

Metrics & Monitoring

Node and pod metrics with CPU, memory, and storage usage. Cluster-wide resource aggregation.

🔍

Custom Resources

Full CRD support with dynamic pages. Automatically discover and display custom resources.

🚀

Easy Deployment

Deploy with Docker, Docker Compose, or Helm. Multi-architecture support (amd64, arm64).

Getting Started

Get up and running in minutes

🐳 Quick Start with Docker Compose

The fastest way to get started:

git clone https://github.com/yourusername/kubelens.git
cd kubelens
./scripts/dev.sh start

# Open http://localhost in your browser

⚓ Deploy with Helm

For production deployments:

helm install kubelens ./charts/kubelens \
--namespace kubelens \
--create-namespace \
--set server.image.tag=latest \
--set app.image.tag=latest

🔨 Build from Source

For development:

# Server
cd src/server
go mod download
go run ./cmd/server/main.go

# App
cd src/app
npm install
npm run dev
